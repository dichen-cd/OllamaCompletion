## Installation

1. Install [Ollama](https://ollama.com/)
2. Download and launch `codellama` by `ollama run codellama`
3. Copy `OllamaCompletion/` into Sublime Text's Packages folder
4. Restart Sublime Text



## Usage

- Just type `ctrl`+`space`
